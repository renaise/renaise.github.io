ai://www. teaching an LLM to reach the live web.

A speculative thesis on how language models touch the open web — with Shovon Hasan.

Prototype

An LLM can describe how to book a flight but can't reach the booking page. The live web sits behind apps and approvals it has no hands for.

The question was blunt: how does a model touch a website when no one has built it an app to do so? Our answer — the model writes an HTTP request as plain text, the browser runs it, and the real site renders inside the chat window. Sites publish a /mythra.txt describing what they can do: a robots.txt for actions instead of crawlers.

This was 2024. A year later the industry called it MCP. The drawing came first.
